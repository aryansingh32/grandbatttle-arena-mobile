# App Icon Directory

## Instructions

1. Place your app icon here: `app_icon.png`
2. Recommended size: **1024x1024 pixels**
3. Format: PNG (with transparency if needed)
4. This icon will be used for:
   - App launcher icon (home screen)
   - App info icon
   - Notification icon
   - All app icons everywhere

## After adding your icon:

1. Update `pubspec.yaml` if your icon has a different name
2. Run: `dart run flutter_launcher_icons`
3. Clean and rebuild: `flutter clean && flutter run`
